# CoverCapture Landing Page

A high-performance, modern landing page for **CoverCapture**, built with Next.js 15, Tailwind CSS, and Framer Motion. This platform focuses on simplifying the commercial insurance coverage lifecycle through a unified, data-driven engine.

## üöÄ Tech Stack

- **Framework:** [Next.js 15](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Components:** Radix UI / Shadcn (Customized)
- **Language:** TypeScript

## ‚ú® Key Features

- **Responsive Design:** Fully optimized for mobile, tablet, and desktop.
- **Dark Mode Support:** Native support for system preferences and manual toggling.
- **Zero Layout Shift:** Custom `HeadlineRotator` with ghost-layer stabilization to prevent content jumping.
- **Fluid Animations:** Staggered entrance animations and spring-physics button interactions.
- **Single Page Experience:** Locked viewport (`100vh`) for a clean, app-like landing experience.
